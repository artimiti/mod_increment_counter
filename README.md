# Increment Counter — модуль для Joomla
Динамический счётчик, который автоматически увеличивает заданное число через указанные промежутки времени.
![Пример работы модуля](https://raw.githubusercontent.com/artimiti/mod_increment_counter/main/browser_X6YhrYm7pv.png)

## Возможности

- Задаётся **стартовое число**
- Указывается **инкремент** (на сколько увеличивать)
- Период срабатывания задаётся **в секундах** — можно использовать арифметические выражения, например:  
  `60 * 60 * 24 * 3` → каждые 3 дня
- Указывается **дата начала отсчёта**
- Расчёт выполняется **в UTC**, независимо от часового пояса сервера
- Поддержка нескольких экземпляров на одной странице

## Установка

1. Скачайте архив [Source code (zip)](https://github.com/artimiti/mod_increment_counter/releases).
2. В админке Joomla: **Система → Установить расширение → Загрузить файл**.
3. После установки создайте экземпляр модуля: **Модули → Создать → Increment Counter**.
4. Настройте параметры и назначьте на нужную позицию.

## Параметры

| Поле | Описание | Пример |
|------|--------|--------|
| **Стартовое число** | Исходное значение счётчика | `3578951` |
| **Инкремент** | На сколько увеличивать за один период | `500` |
| **Период (выражение в секундах)** | Интервал между увеличениями. Можно использовать `*`, `+`, скобки. | `60*60*24*7` (раз в неделю) |
| **Дата начала отсчёта** | Дата, с которой начинается отсчёт (время игнорируется, используется 00:00 UTC) | `2026-01-31` |

## Пример работы

- Стартовое число: `3578951`
- Инкремент: `500`
- Период: `60*60*24*7` (7 дней = 604800 сек)
- Дата начала: `2026-01-12`

Результат:
- 12.01.2026 → `3578951`
- 19.01.2026 → `3579451`
- 26.01.2026 → `3579951`
